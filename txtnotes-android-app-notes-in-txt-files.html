TXTnotes - offline free Android app saves notes locally in .TXT files and supports microSD

Download the app installation APK file: https://github.com/Eb43/txtnotes/blob/main/TXTNotes.apk


Make Android notes in tXT files and save them anywhere, including external microSD card. Transfer Android notes in a standard TXT format supported by Windows, MacOS, Linux, ChromeOS, Android, iOS.

Choose a folder for the app to read and save TXT files. Edit any existing TXT file in that folder.

App displays files sorted by last modification dated.

File names for files created by the app are assigned in the format DayMonthYearHourMinuteSecond.TXT

Each file is shown as a card with file's name and a preview of its first several hundred characters

To create a new file click + button. To rescan the selected folder press refresh button.

To open a file shown in the list, click on it. To exit the file editing screen press X button.


  • Stores your notes in .TXT files. One note - one file.
  • Saves files in user-specified folder. One folder - one set of notes.
  • Supports saving on microSD card using native Android file access framework.
  • Automatically sorts the list of files within the folder by last modified date.
  • Prevents unsaved note loss with automated save on exit or app minimization.
  • No hidden metadata and databases that are inaccessible to users. Only TXT files with filenames in the format of file creation date.
  • Displays a snippet of TXT file content for easy and quick content preview.
  • No tracking, no ads, no Google Play Services, fully offline.
  • Free to use. No in-app purchases, no limitations, no accounts.
  • Minimal system requirements. No animations, no background services.
  • Only 6.5 MB of APK installer file.
  • Supports modern Android versions starting from Android 8.
  • Open source code.
